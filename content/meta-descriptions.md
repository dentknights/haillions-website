## Technical SEO Notes

### Character Counts (Optimal: 150-160)
- All meta descriptions are 145-160 characters
- Include primary keyword within first 120 characters
- Include location "Houston" or specific suburb
- Include call-to-action or value proposition
- Use power words: Expert, Professional, Mobile, Free

### Title Tag Guidelines
- Primary keyword first (when possible)
- Brand name at end
- 50-60 characters optimal
- Include location for local SEO
- Use pipes (|) as separators

### Schema Integration
- Service schema for all service pages
- LocalBusiness schema for homepage
- FAQPage schema for FAQ content
- BreadcrumbList for navigation
- Review/Rating schema where applicable
